# FAQ

## What is Conductor?

Conductor is planned as a .NET application for supervising many Symphony instances from one operational dashboard.

## Where are the requirements?

The requirements are maintained in [requirements.md](requirements.md). The implementation plan is maintained in [technical.md](technical.md).

## How are documentation checks run?

Run the local validator:

```powershell
./scripts/validate-docs.ps1
```
