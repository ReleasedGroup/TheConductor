# Platform Rules

This directory contains rules for building and testing platform-specific code.

It contains folders for each major technology stack and AGENTS.md and other documentation files that provide guidelines and best practices for developing platform-specific code.

# Directory Structure

- `platform-rules/`: The root directory for platform-specific rules.
- `platform-rules/AGENTS.md`: Documentation file that provides an overview of the platform-specific rules and guidelines.
- `platform-rules/README.md`: A readme file that provides an overview of the platform-specific rules and guidelines.
- `platform-rules/<technology-stack>/`: Subdirectories for each major technology stack (e.g., `java`, `python`, `javascript`, etc.) that contain platform-specific rules and guidelines.
- `platform-rules/<technology-stack>/AGENTS.md`: A markdown file that contains the rules and guidelines for the specific technology stack.
